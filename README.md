# re-japan-todo

日本が先送りせず解くべき課題を、AIが出した結論としてではなく、人間が検証できる「課題地図」として読むためのUIプロトタイプです。

このリポジトリは、深津 貴之さんの以下のページへのフィードバック用に作成した実験版です。

- 元ページ: https://fladdict.github.io/japan-todo/

## コンセプト

各課題カードに、以下の4モジュールを追加しています。

1. 信頼度メーター
2. 根拠・出典レイヤー
3. 反対論・副作用レイヤー
4. 次アクション・検証レイヤー

AIが社会課題の「答え」を出すのではなく、人間が次に調べるべき問いを見つけるための入口にすることを目的にしています。

## 使い方

- 検索欄で課題名やキーワードを絞り込み
- 信頼度フィルタでカードを分類
- 各カードを開閉して、根拠・反対論・次アクションを確認
